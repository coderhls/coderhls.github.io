变量与常量

在Dart中，可以声明强类型与弱类型两种变量。因此，在学习之前，需要先了解一下强类型与弱类型的变量

引用知乎文章：https://zhuanlan.zhihu.com/p/62570358

变量

弱类型

var

如果没有初始化变量，那么该变量可以变成任何类型

dynamic

变量可以为任意类型，编译阶段不检查类型

Object

变量可以为任意类型，编译阶段检查类型

区别

从上述例子中也能够看出，如果var在声明变量时初始化，那么类型将被锁定，而dynamic与Object则不需要有这个顾虑

强类型

明确声明变量的方式，定义的变量可以修改值, 但是不能赋值其他类型

名称 说明
num 数字
int 整型
double 浮点
bool 布尔
String 字符串
StringBuffer 字符串 buffer
DateTime 时间日期
List 列表
Set 无重复队列
Map kv 容器
enum 枚举

变量的默认值

Dart也是面向对象的语言，在面向对象中，万事万物皆对象，因此变量的默认值都是null

常量

在Dart中除了变量的定义外，也同样可以定义常量。final和const都是用于定义常量的, 也就是定义之后值都不可以修改

类型声明可以省略

由于常量的必须在定义时初始化，因此可以省略类型的声明

初始化后不能再修改值

这是一个众所周知的规定，常量在初始化值后不能再修改值。

不能与var一起使用

区别

  • const在赋值时, 赋值的内容必须是在编译期间就确定下来的
  • final在赋值时, 可以动态获取, 比如赋值一个函数
